How to participate?

  1. Join to the CATJS Linkedin group
  2. Follow the instructions in the CATJS npm project and install the tool.
  3. Write your scraps (tests).
    You can get more information about how to write CATJS scrap via below links:
    • CATJS site
    • Technology Through My Eyes blog
    • Method and Tools
    • CATJS video training
  4. Submit a short screen captured video of your running scraps along with written a description (3 paragraphs) of your work to the CATJS linkedin group.
  5. Invite people to "like" your post.

CATJS is an opensource automation framework for testing web and mobile-web application.
For more infromation about CATJS visit CATJS site.
