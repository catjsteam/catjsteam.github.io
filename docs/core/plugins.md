CatJS
==============

## Plugins
  CatJS has plugable solution for both Annotations and Tasks. 


* [Annotations Plugins](plugins/annotation_plugins.html)
The annotation technology is plugable. There are OOTB plugins that are built-in and can be used and there's an option to create a custom annotation plugin.


* [Tasks Plugins](plugins/task_plugins.html)
CatJS Task Runner is plugin based. There are OOTB plugins that are built-in and can be used and there's an option to create a custom Task plugin
